# Plotcraft

Crafting publication-ready statistical graphics in Python.

Plotcraft is a Python data visualization library for building elegant, publication-ready statistical graphics from tidy data. Inspired by the principles of the Grammar of Graphics and the design of the tidyplot ecosystem, Plotcraft provides a clean, declarative, and chainable API for constructing complex visualizations in a structured and intuitive way.

With Plotcraft, plots are built layer by layer using a fluent method-chaining interface. This approach separates data, aesthetic mappings, and visual layers, making it easy to create clear, reproducible, and customizable figures suitable for scientific publications.

Plotcraft is designed to streamline the creation of high-quality graphics for research papers, reports, and exploratory data analysis while remaining flexible enough for advanced visualization workflows.
